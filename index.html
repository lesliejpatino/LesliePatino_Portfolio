Password Generator

This website generates a custom password for the user dependent on their preferences. At minimum, the user must select the length of their password and one criteria such as lower or uppercase letters, numbers or special characters.
Once the criteria is selected, the generator displays a password for the user. The generator will display error messages if the user does not select at least one criteria for their password.
